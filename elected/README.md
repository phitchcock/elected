## Developer Documentation

#### Requirements
* chruby
* Ruby Version 2.3.0
* Rails 4.2.5

#### Setup
1. Download the repo
2. Run `bundle install` or `bundle update`
3. Start the rails server `rails s`
4. Jump on `localhost:3000` with your browser
